Collage

How I did it:

Create a new layer and fill it with the main mountain image first.
• Next, use the ellips tool to cut the saved yoga image into an oval shape on a new transparent layer before adding the feather effect.
• Make a copy of the image of the obelisk and paste it on the layer of the mountain while setting the yoga image's opacity to 60%.
• Add a second saved image of the village with a size of 300 * 90 pixels to the collage file.
• Next, create a font box using the text tool and enter your MIT ID in it. Then set the text in the example file to the appropriate font-size of 42 serif bold.
• Check the size of the entire canvas at the end and fix it to 600 * 400 pixels.
